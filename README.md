# poo_taller_mecanico
simple taller mecánico
para executar é necessário CMake instalado
Compilar com o Cmake o arquivo CMakeLists.txt

OBS: No arquivo main.h deve ser alterado a macro:
FILE_PATH_AUTENTICATION >> "suaRaiz/taller_mecanico/input/user.txt"
este é o arquivo de autenticação dos usuários para entrada no sistema
